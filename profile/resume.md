## Стиль сопроводительных писем

- Язык: русский
- Начинать с «Здравствуйте!»
- Длина: 5-8 коротких абзацев, 1200-2200 символов для релевантной вакансии; 600-1000 символов для слабо подходящей
- Формат: готово для вставки в отклик HH.ru
- Без подписи «С уважением»
- Без плейсхолдеров типа «[Ваше имя]» или «[название компании]»
- Не выдумывать годы опыта, компании и навыки, которых нет в профиле
- Не писать «более 5 лет опыта» и не добавлять выдуманную статистику
- Не использовать длинное тире (—)
- Выбирать главный проект, наиболее релевантный вакансии, и описывать его конкретно
- Писать конкретно, без общих AI-клише

### Структура письма

1. «Здравствуйте!»
2. Почему вакансия заинтересовала: 1 абзац с 1-2 конкретными сигналами из вакансии
3. Главный релевантный проект (bot-mont-shk для WB/e-commerce/аналитики; AI-проекты для AI/LLM/агентов)
4. Дополнительные AI-кейсы, если вакансия про AI/LLM/автоматизацию
5. Стек — только то, что совпадает с вакансией
6. Рабочий подход: разобраться в процессе, найти узкие места, внедрить 2-3 быстрых автоматизации
7. Ссылки на GitHub и сайт, если уместно
